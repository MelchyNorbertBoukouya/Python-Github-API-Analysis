# Python-Github-API-Analysis


## Project Overview

This project demonstrates how to collect real-world data from the GitHub REST API and perform Exploratory Data Analysis (EDA) using Python.

The project covers the complete data analysis workflow:

* Collecting data through an API
* Working with JSON responses
* Creating a Pandas DataFrame
* Cleaning and exploring the dataset
* Performing statistical analysis
* Performing vizualisations

## Technologies Used

* Python
* Requests
* Pandas
* NumPy
* Jupyter Notebook
* Matplotlib

## Dataset

The dataset is collected directly from the GitHub REST API by searching for the most popular Python repositories.

## Analysis Performed

* Repository ranking by stars
* Average number of stars
* Average number of forks
* Correlation between stars and forks
* Missing value analysis
* visualizations

## Project Workflow

GitHub API → JSON → Pandas → Data Cleaning → EDA → Visualization
